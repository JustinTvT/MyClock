## Use
app:clock_calibration_color="#ccff9900"//时钟刻度颜色值
app:clock_hour_color="#9fe3f6"//时针颜色
app:clock_minute_color="#cc0000"//分针颜色
app:clock_second_color="#0099FF"//秒针颜色
